4.1 より厳密な配布物作成手順
--------------------------
```bash
# Git履歴からも完全除外（必要に応じて）
git rm --cached postgres_password.txt

# .gitignore確認
echo "postgres_password.txt" >> .gitignore

# クリーンな配布用アーカイブ作成
zip -r rakuten_v3.1_clean.zip rakuten/ -x "*postgres_password.txt*"
```

4.2 本番環境での推奨設定
----------------------
```bash
# より強固なパスワード生成
openssl rand -base64 32 > postgres_password.txt

# 厳密なファイル権限
chmod 600 postgres_password.txt
chown root:root postgres_password.txt  # 必要に応じて
```

4.3 自動化スクリプト例
--------------------
```bash
#!/bin/bash
# setup_secrets.sh
echo "Setting up PostgreSQL secrets..."
if [ ! -f postgres_password.txt ]; then
    echo "$(openssl rand -base64 32)" > postgres_password.txt
    chmod 600 postgres_password.txt
    echo "✅ postgres_password.txt created with secure password"
else
    echo "⚠️ postgres_password.txt already exists"
fi
```
