## 2. 作業指示

### 2-A. CI ワークフロー
`/.github/workflows/ci.yml` を新規作成：

```yaml
name: CI
on: [push, pull_request]
jobs:
  lint-test:
    runs-on: self-hosted
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with: { python-version: '3.12' }
      - run: |
          python -m pip install --upgrade pip
          pip install -r requirements.txt
      - run: ruff check .
      - run: flake8 .
      - run: pytest -q
留意: PAT が不要なら token: オプションは書かない。

/opt/rakutenApp/rakuten-monitor.service をリポジトリ直下に置く:

ini

[Unit]
Description=Rakuten Monitor
After=network.target

[Service]
User=yang_server
WorkingDirectory=/opt/rakutenApp
EnvironmentFile=/opt/rakutenApp/.env
ExecStart=/opt/rakutenApp/.venv/bin/python -m monitor --cron
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
